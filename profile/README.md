# Motion — Portable Motion Graphics & 2D/3D Animation

## About Motion

Motion provides a complete motion graphics and animation environment for video editors and motion designers. Motion combines behaviors, keyframes, particles, replicators, 3D text, and cameras. Motion enables users to create titles, lower thirds, transitions, effects, and animated graphics. Motion runs on desktop platforms. Motion is trusted by video editors, motion designers, YouTubers, broadcast professionals, and Final Cut Pro users worldwide.

## Deep Dive — Motion Graphics Notes

Behaviors: automatic animations (gravity, orbit, fade, rotate, throw)
Keyframes: manual animation, curve editor, interpolation (bezier, continuous)
Particles: emitters, particle types, forces, physics, 2D/3D particles
Replicators: pattern duplication, grids, circles, spirals, image patterns
3D: 3D text, cameras, lights, depth of field, 3D particles
